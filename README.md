# jc.riv-website — Artist Portfolio

Static multi-page website for an artist. Built with raw HTML + CSS, no frameworks. Deployed on GitHub Pages.

## Project Structure

```
├── index.html              # Landing page + CV
├── styles.css              # Global CSS
├── AGENTS.md               # AI agent instructions
├── .opencode/skills/       # opencode skills
├── series/                 # Art series sub-pages
├── courses/                # Painting courses page
└── assets/images/          # Grouped by series
```

## Quick Start

Open `index.html` in your browser to preview. No build step required.
